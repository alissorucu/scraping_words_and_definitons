# scraping_words_and_definitons

## Web Scraping with Python Selenium

for my flutter project I need a lot of questions and I could not write them manually.
To solve this problem I used Python-Selenium.

<br/><br/>
#### My Question Class
```
class Question {
  String defination;
  int trueIndex;
  List<String> answers;
  Question(this.trueIndex, this.defination, this.answers);
}
```
